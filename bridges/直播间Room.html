# 直播间 Room

# 获取房间(主播)信息 getRoomInfo

API 介绍

获取主播信息

API 原型

DYBridge.dispatch('Room','getRoomInfo',{},funtion(error,msg,data){

})

参数说明

无

回调参数说明

data：

{
    "roomId":"",//房间id
    "cateId":"",//二级分类id
    "avatar":"",//头像
    "nickname":"",//昵称
    "anchorUid":"",//主播id
}

示例

DYBridge.dispatch('Room','getRoomInfo',{},funtion(error,msg,data){
	if(error==0){
    	let roomId=data.roomId;
    	
	}
})

# 切换房间 changeRoom

API 介绍

同类型直播间切换房间

API 原型

DYBridge.dispatch('Room','changeRoom',json,function(error,msg,data){
    
})

json参数说明

{
	"roomId":"";//String类型，要跳转的直播间id
}

回调参数说明

无

示例

DYBridge.dispatch('Room','changeRoom',{'roomId':'101'},function(){})

# 发送弹幕消息 sendCppMsg

API 介绍

发送弹幕

API 原型

DYBridge.dispatch('Room','sendCppMsg',json,function(error,msg,data){})

参数说明

json为一个JSON对象,具体的字段及含义需要参考C++消息文档

回调参数说明

无

示例

DYBridge.dispatch('Room','sendCppMsg',{
    "type":"chatmessage",
    "content":"6666",
    "pid":"0"
},null)
