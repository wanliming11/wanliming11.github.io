# 分享 share

API 介绍

弹出分享框。原有的setShareInfo 取消，可以采用把分享按钮点击事件回调给前端，前端再调该方法

API 原型

DYBridge.dispatch('Share','share',json,function(error,msg,data){})

json参数说明

{
	"name":"",//String 类型,分享的标题
    "content":"",//String 类型,分享的内容
    "url":"",//String类型，分享的图片地址
    "link":"",//String类型，分享的链接地址
    "platform":"",//String 类型，分享的平台  0==所有,1==微信，2==朋友圈，3==QQ空间，4==微博，5==QQ，6== 鱼吧，7== 截屏分享。 选中多个平台中间用逗号隔开，如`1,2,3,4` 
    "yubadata":"",//String 类型，如果选择鱼吧,则进行参数透传
    "from":"",//String 类型，
    "scrsurl":"",//String类型，分享图片的url地址 scrs=screenshots的缩写
    "scrstitle":"",//String类型，截屏分享 分享标题
    "scrsdesc":"",//String类型，截屏分享 分享描述
    "imgBase64":"",//String类型 base64位图片字符
    "type":0//number类型 0表示分享链接，1表示分享图片
}

回调参数说明

无

示例

DYBridge.dispatch('Share','share',{
    "name":"分享标题",
    "content":"这是分享内容",
    "link":"https://m.douyu.com",
    "platform":"1"
},function(error,msg,data){

})

# 绑定平台 bindPlatform

API 介绍

绑定平台。

API 原型

DYBridge.dispatch('Share','bindPlatform',json,function(error,msg,data){})

json参数说明

{
    "type":1//number类型 表示社交平台类型,1=新浪, 2=微信, 3=QQ
}

回调参数说明

error : 绑定状态码 msg: 绑定提示信息

示例

DYBridge.dispatch('Share','bindPlatform',{type:1},function(error,msg,data){

})

# 跳转到微信小程序 gotoWXmini

API 介绍

API 原型

DYBridge.dispatch('Share','gotoWXmini',json,function(error,msg,data){})

json参数说明

{
	"userName":"",//拉起的小程序的username
	"path":"",//拉起小程序页面的可带参路径
}

回调参数说明

无

示例

DYBridge.dispatch('Share','gotoWXmini',{
    "userName":"gh_2f0478fcecb6",
    "path":"pages/activities/make-card?from=zhongjianye"
},function(error,msg,data){

})
